# Стартовый промпт для реализации фичи «Боковая панель с деревом файлов»

Скопируй целиком и вставь в новую сессию.

---

Реализуй фичу **Боковая панель с деревом файлов** по подготовленной документации.

## Документация

- Спецификация: `docs/features/sidebar-file-tree/spec.md` — прочитай перед началом
- План: `docs/features/sidebar-file-tree/plan.md` — следуй порядку задач, секции «Стратегия выполнения» и «Ревью после каждого шага»
- Чеклист: `docs/features/sidebar-file-tree/checklist.md` — отмечай прогресс

## Инструкции

1. Прочитай spec.md, plan.md и checklist.md целиком.
2. Создай ветку `feature/sidebar-file-tree` от `main` и переключись на неё.
3. Отметь в checklist.md пункты «Подготовка» как выполненные.
4. Выполняй задачи согласно **стратегии выполнения** и зависимостям в plan.md:
   - `sequential` — строго по порядку
   - `parallel-subagent` — можно вынести параллельным субагентам после выполнения зависимостей
   - `parallel-same` — параллельно в одной сессии, если инструмент позволяет
5. После **каждой** задачи: ревью по plan.md и spec.md; убедись, что нет конфликта с параллельными задачами (общие файлы, противоречивая логика). Если задачу делал субагент — проверь результат перед следующим шагом.
6. После выполнения задачи — отметь её в checklist.md (`[x]`).
7. Запусти указанную для задачи проверку.
8. Коммить после каждого логического блока задач.
9. По завершении — отметь «Финализация» в checklist.md и обнови статус в README.md на `Done`.
10. Если что-то непонятно — спроси, не додумывай.
