# Стартовый промпт для реализации фичи «Инициализация проекта»

Скопируй целиком и вставь в новую сессию.

---

Реализуй фичу **Инициализация проекта** по подготовленной документации.

## Документация

- Спецификация: `docs/features/init/spec.md` — прочитай перед началом
- План: `docs/features/init/plan.md` — следуй порядку задач, секции «Стратегия выполнения» и «Ревью после каждого шага»
- Чеклист: `docs/features/init/checklist.md` — отмечай прогресс
- Родительский план (для контекста): `docs/plans/2026-04-15-terminal-wrapper-stage1.md` — Task 1

## Инструкции

1. Прочитай spec.md, plan.md и checklist.md целиком.
2. Отдельную ветку не создавай (см. README.md). Git будет инициализирован в рамках задачи #6.
3. Отметь в checklist.md пункты «Подготовка» как выполненные.
4. Выполняй задачи согласно **стратегии выполнения** и зависимостям в plan.md:
   - Задачи #1 и #3 можно создать параллельно (разные файлы).
   - Задача #2 — после #1.
   - Задача #4 (`npm install`) — после #1.
   - Задача #5 — можно параллельно с #4.
   - Задача #6 — последняя, после всех.
5. После **каждой** задачи: ревью по plan.md и spec.md; убедись, что нет конфликта с параллельными задачами.
6. После выполнения задачи — отметь её в checklist.md (`[x]`).
7. Запусти указанную для задачи проверку.
8. **Важно для задачи #4:** если `electron-rebuild` падает — проверь, что установлены Xcode Command Line Tools (`xcode-select --install`).
9. По завершении — отметь «Финализация» в checklist.md и обнови статус в README.md на `Done`.
10. Если что-то непонятно — спроси, не додумывай.
