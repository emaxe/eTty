# Стартовый промпт

Реализуй фичу **Сохранение состояния вкладок** по подготовленной документации.

## Документация

- Спецификация: `docs/features/tab-persistence/spec.md` — прочитай перед началом
- План: `docs/features/tab-persistence/plan.md` — следуй порядку задач
- Чеклист: `docs/features/tab-persistence/checklist.md` — отмечай прогресс

## Инструкции

1. Прочитай spec.md и plan.md целиком
2. Отдельную ветку не создавай (см. README.md)
3. Отметь в checklist.md пункты «Подготовка» как выполненные
4. Выполняй задачи согласно стратегии выполнения в plan.md
5. После каждой задачи — ревью по plan.md и spec.md
6. После выполнения задачи — отметь её в checklist.md (`[x]`)
7. Запусти `npm run dev` для проверки
8. Коммить после завершения блока задач
9. По завершении — отметь «Финализация» в checklist.md и обнови статус в README.md на `Done`
